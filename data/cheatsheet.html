Location: /Users/tp/Projects/Catch

1) Activate venv

cd /Users/tp/Projects/Catch
  source .venv/bin/activate

2) Install Playwright (reference)

python -m pip install --upgrade pip
  python -m pip install playwright
  python -m playwright install chromium

3) Run scraper (all seven stores)

python scripts/scrape_revel_playwright.py --csv --download-images

Outputs: data/revel/<store-slug>/ (JSON, CSV, images/)

4) Skip Post Oak (already scraped)

python scripts/scrape_revel_playwright.py --exclude store-5-post-oak --csv --download-images

Exclude by ID: --exclude 5; single store: --store store-105

5) Headed mode

python scripts/scrape_revel_playwright.py --headed

6) Deactivate venv

deactivate

Notes

  • Script closes browser/page and disposes the API context automatically.
  • Playwright binaries cache: ~/Library/Caches/ms-playwright (optional to prune).
